# Retail-Sales-Performance-Analytics
This project presents a complete end-to-end data analytics solution for a retail business, transforming raw transactional data into actionable insights and an executive-level dashboard.
The goal is to simulate a real-world business scenario where data is used to drive decisions, optimize performance, and improve profitability.

# Business Objectives
*Analyze sales and profit performance across regions
*Identify top-performing categories and products
*Evaluate the impact of discounts on profitability
*Discover high-value customers and cities
*Provide a centralized dashboard for decision-makers

# Tech Stack
*Excel - Data Cleaning & Preprocessing
*SQL (MySQL)	- Data Analysis & Querying
*Power BI	- Data Visualization & Dashboard

# Project Workflow
1. Data Cleaning (Excel)
*Handled missing and inconsistent values
*Standardized categorical fields (Region, Segment, Category)
*Converted columns into proper data types
*Ensured dataset readiness for analysis
2. Data Analysis (SQL)
*Performed business-driven analysis using SQL
*Used advanced concepts: GROUP BY, ORDER BY, CASE WHEN, Aggregations (SUM, AVG, COUNT)
*Extracted insights like : Region-wise Sales & Profit, Category performance, Discount vs Profit relationship, Top customers & cities
3. Dashboard Development (Power BI)
*Designed an interactive executive dashboard
*Key components:
  - KPI Cards → Total Sales, Total Profit, Profit Margin
  - Region-wise performance analysis
  - Category-wise profitability
  - Discount vs Profit scatter analysis
  - Top Cities & Customers insights
  - Added slicers for dynamic filtering

# Key Insights
* West Region leads in overall sales
* Technology category generates highest profit
* High discounts negatively impact profitability
* Top customers contribute significantly to revenue
*  Some regions show high sales but low profit margins

# Business Impact

* Enabled identification of profitable vs loss-making segments
* Improved understanding of discount strategies
* Helped in targeting high-value customers
* Delivered a single source of truth dashboard for stakeholders
